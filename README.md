# CKD Diagnosis using EFJMI + CSA-GA Feature Selection

## Overview
This repository contains a complete implementation of an Enhanced Fuzzy Joint Mutual Information (EFJMI) feature-selection framework combined with Cuckoo Search Algorithm (CSA) and Genetic Algorithm (GA) optimization for Chronic Kidney Disease (CKD) diagnosis.

## Features
- CKD dataset preprocessing and cleaning
- Missing-value imputation
- IQR-based outlier handling
- EFJMI feature ranking
- CSA-GA feature subset optimization
- Baseline feature-selection comparisons
- Machine learning model evaluation
- Explainability using SHAP

## Dataset
Use the UCI Chronic Kidney Disease (CKD) dataset in ARFF format.
